# Installation Dependencies:
- python3
- pytorch
- pygame
- numpy
- opencv

# How to Run?
## Training:  
set ``train = True`` in DDQN_with_Pong.py and run with following command:  
```python DDQN_with_Pong.py```
## Inference:  #
set ``train = False`` in DDQN_with_Pong.py and run with following command:  
```python DDQN_with_Pong.py```
