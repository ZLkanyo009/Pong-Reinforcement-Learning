- python3

- pygame

# How to Run?
## Training:  
set ``istrain = True`` in deep_q_network.py and run with following command:  
```python deep_q_network.py```
## Inference:  #
set ``istrain = False`` in deep_q_network.py and run with following command:  
```python deep_q_network.py```
